Nuestro trabajo

Utilizamos Arduino y Software especializado

Arduino

Arduino es una placa electrónica de hardaware libre, la cual mediante el uso sensores, podemos controlar los momentos justos para que se activen las bombas para el riego de cultivos.

Software especializado

Con el software especializado, usted puede obtener el control del las acciones que la placa realiza, como por ejemplo, modificar las horas de riego, cuanto tiempo desea que se realize el riego, entre muchas otras cosas más.
